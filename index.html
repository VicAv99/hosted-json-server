Deploy to now

  1. [Create your database](#create-your-database)
  2. Install now cli-tool globally npm install -g now
  3. Run the `now` command in this folder/repo where your project is. If you run it for the first time, you will be prompted to login, after login, run the command again: now --public _--public is to skip the prompt telling you that you will open source your project if you deploy it to now_
  4. The URL will be copied automatically and you can just paste it into your browser.
  5. **Optional**: Rename the deployment: now alias https://your-deployed-name.now.sh new-name _first argument is the deployed site, second argument is the new name to give it_
